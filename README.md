Wiki介绍
【由ZZQLittleBean与Kimi AI共同开发！】一款实验性的Wiki网站，当前存储《另我多才多艺》及其衍生作《向向少女不得闲，晒足一百八十天》的内容，后续将上线空白Wiki的GitHub与本地版供用户编辑。网站当前的信息来源是重制版的《另我多才多艺》内容，可能会与原版有少许出入。后续衍生作的内容会正式加入其中。
Wiki的特色功能：角色与设定的词条（可包含多个词条版本，每个版本可进行单独编辑但最终会绑定一个编号，适合添加一个角色的多种形态）
由于GitHub版本运行环境限制，绝大多数功能在该版本进行精简，后续可期待从Wiki-Is-She-Also-me-Who-is-So-Cute-ZZQLTB-Local更新的本地版Wiki，包括更多的功能：
阵营划分与角色关系
更具有特色的角色关系图
更多的词条编辑功能，包括关系/相关角色索引
更实用的时间轴类词条筛选与更美观的剧情梗概功能
GitHub的可部署版本在打磨完全后也会择机上线。
由于内容大量采用与AI合作的形式达成，初级版本难免会出现甚至复现各种各样不可名状的Bug。共建Wiki功能及反馈Bug，请联系电子邮箱：zzqlittlebean@icloud.com 。
作品介绍
Wiki的内容将会随着《另我多才多艺》重制版进度的更新而更新。内容中
《另》采用旁白与第一人称交叉形式，重点描摹主角张可云经历“世界融合”变故后，与“另一个自己”张可颂的日常生活。
作品特色：
性转/TS：作品旨在从多样的性别方向思考生活与世界。可能会存在少量「让人血脉喷张」的情节，但请注意完全不存在涉及暴露的色情内容。
校园与魔法：作品与作者的校园经历有不少关联，某种意义上算是一种特殊的“自传”。在部分时间点会有*「奇迹」的出现：*除了TS题材，作者虚构的内容在未来一定时间内发生的概率极高，毒奶率极高，但仅供娱乐。作品中有轻微的魔法类设定，但绝大多数内容仍然基于现实。
